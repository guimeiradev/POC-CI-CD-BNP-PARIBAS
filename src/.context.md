The .NET 10 solution built, tested, and published by the root `Jenkinsfile`
pipeline: a minimal web API and its test project.

## Summary

- BnpPoc.sln — Visual Studio solution referencing `BnpPoc.Api` and `BnpPoc.Api.Tests`; both are restored/built/tested via `dotnet <cmd> src/BnpPoc.sln` in the Jenkins pipeline.
- BnpPoc.Api/ — the ASP.NET Core Minimal API application (the artifact that gets published and uploaded to Nexus).
- BnpPoc.Api.Tests/ — xUnit integration tests for `BnpPoc.Api`.

## Constraints

- The solution file's project names and GUIDs MUST stay in sync with the two `.csproj` files it references — the Jenkinsfile builds/tests via the `.sln`, not individual projects.
